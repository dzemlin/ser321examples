You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /rollDice -- rolls a specifide number of polyhedral dice
  • ---- example: /rollDice?quantity=2&sides=6
  • /badBoy -- Makes the bad boy write on the black board a chosen word, a given number of times
  • ---- example: /badBoy?word=good×=1000
